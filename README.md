## Réponses aux questions

#### Pourquoi l'attribut graine est final ?
L'attribut graine est final car il n'est pas emmené à être modifié une fois qu'il est passé dans 
le constructeur.

#### Quel type utilisez-vous ?
J'utilise le type Double.

#### Pourquoi sortir, selon vous, ce bout d'algorithme de la classe ?
Sortir ce bout d'algorithme permet de faciliter l'ajout d'un nouveau type de terrain
plus facilement si besoin.
